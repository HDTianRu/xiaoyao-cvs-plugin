# xiaoyao-cvs-plugin

改版自用，去除图鉴功能，保留sk，ck等功能

## 安装
使用gitee
```sh
git clone https://gitee.com/HDTianRu/xiaoyao-cvs-plugin.git ./plugins/xiaoyao-cvs-plugin/
```
使用github
```sh
git clone https://github.com/HDTianRu/xiaoyao-cvs-plugin.git ./plugins/xiaoyao-cvs-plugin/
```
